# Dev Env setup

- Download .NET core
- check out the repository
- switch into the directory
- run `dotnet restore`
- run `dotnet ef database update`
- run `dotnet run` 
- or  `dotnet watch run` for server rebuild on save
- visit website on localhost:5000
